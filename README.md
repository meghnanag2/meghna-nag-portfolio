- `meghna_nag_resume.pdf` — Resume (downloadable)
- `meghna_professional_photo.png` — Professional headshot
- Award images and certificates
- Training & academic certificates

## Deployment

### GoDaddy (FTP Upload)
1. Download all files from this repository
2. Connect to your GoDaddy cPanel
3. Open File Manager → public_html
4. Upload all files

Your site will be live at `yourdomainname.com`

### GitHub Pages
1. Create a GitHub repository
2. Push these files
3. Go to Settings → Pages
4. Select "Deploy from branch" (main)
5. Your site will be live at `username.github.io/portfolio`
